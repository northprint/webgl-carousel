WebGL Carousel - Basic Usage Example

About This Example

This demonstrates basic WebGL Carousel usage with:

  • Simple initialization
  • Navigation controls (Previous/Next buttons)
  • Play/Pause functionality
  • Default fade effect
  • Status display

Code Snippet

const carousel = new WebGLCarousel.WebGLCarousel({
  container: '#carousel',
  images: images,
  effect: 'fade',
  transitionDuration: 1500,
  autoplay: false,
  loop: true
});
