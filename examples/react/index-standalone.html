React Integration Features

This standalone demo simulates what the React component provides:

  • Declarative component API
  • State management with hooks (useState, useCallback)
  • Event callbacks for all carousel events
  • Dynamic prop updates that trigger re-renders
  • Full TypeScript support with type definitions
