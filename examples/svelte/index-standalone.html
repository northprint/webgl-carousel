Svelte Integration Features

This standalone demo simulates what the Svelte component provides:

  • Native Svelte component with reactive bindings
  • Event dispatching for carousel events
  • Two-way data binding with bind: directive
  • TypeScript support out of the box
  • Svelte stores for state management

Svelte Component Usage

<script>
  import { WebGLCarouselSvelte } from 'webgl-carousel/svelte';
  
  const images = ['image1.jpg', 'image2.jpg'];
  let selectedEffect = 'fade';
  
  function handleImageChange(event) {
    console.log('Changed to image:', event.detail);
  }
</script>

<WebGLCarouselSvelte
  {images}
  effect={selectedEffect}
  autoplay={true}
  transitionDuration={1500}
  on:imageChange={handleImageChange}
/>

Running the Real Svelte Example

To run the actual Svelte component example:

cd examples/svelte
npm install
npm run dev

Then open http://localhost:5173 in your browser.
