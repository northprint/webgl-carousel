Vue 3 Integration Features

This standalone demo simulates what the Vue 3 component provides:

  • Vue 3 Composition API with reactive refs
  • Reactive state management with ref() and reactive()
  • Event handling with @ syntax (@image-change, @transition-start)
  • Two-way binding with v-model directive
  • Full TypeScript support with Vue 3 type definitions

Vue 3 Component Usage

<template>
  <WebGLCarouselVue
    :images="images"
    :effect="selectedEffect"
    :autoplay="true"
    :transition-duration="1500"
    @image-change="handleImageChange"
  />
</template>

<script setup>
import { ref } from 'vue';
import { WebGLCarouselVue } from 'webgl-carousel/vue';

const images = ['image1.jpg', 'image2.jpg'];
const selectedEffect = ref('fade');

const handleImageChange = (index) => {
  console.log('Changed to image:', index);
};
</script>

Running the Real Vue Example

To run the actual Vue 3 component example:

cd examples/vue
npm install
npm run dev

Then open http://localhost:5173 in your browser.
